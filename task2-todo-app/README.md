# InterSpark Internship - Mini JS App Project

## Project Title: Vibrant Minimalist To-Do List

### Project Goal
To build an interactive task manager application focusing on clean web presentation and functional JavaScript Document Object Model (DOM) manipulation.

### Core Features Implemented
* **Dynamic Task Creation:** Users can input a task and press "Add" or hit the "Enter" key to append an item to the list.
* **Interactive State Toggling (Update):** Clicking a task item dynamically registers an event listener that applies a visual cross-out effect and transitions the card layout to a soft lavender hue.
* **DOM Node Removal (Delete):** A dedicated removal handler target-deletes individual tasks entirely out of the active DOM tree structure.
* **Input Validation:** Built-in validation structural check prevents empty text inputs from creating placeholder lists.

### Tech Stack Utilized
* **HTML5:** Structuring the core container, semantic inputs, and list placeholders.
* **CSS3:** Presenting a clean, vibrant gradient canvas with responsive container layouts and dynamic class state transitions.
* **JavaScript (Vanilla ES6):** Event listening, active node creation, text parsing, and DOM modification methods (`getElementById`, `createElement`, `appendChild`, `toggle`, `remove`).
