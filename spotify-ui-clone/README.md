# Spotify UI Clone 🎧⚡

A pixel-perfect, highly responsive front-end replication of the Spotify desktop application interface. This project was developed as **Task 3** for the **InternSpark Frontend Developer Virtual Internship**.

The core objective of this task was to master advanced layout orchestration, maintain visual hierarchy under dark-mode constraints, and implement interactive micro-animations that mirror a world-class production user interface.

---

## ✨ Features & Layout Highlights

* **Advanced Layout Architecture:** Built using a hybrid of CSS Grid and Flexbox to deliver a seamless three-column layout (Fixed Sidebar Navigation, Main Content Stream, and Dynamic Library panel).
* **Persistent Media Control Player:** A sticky bottom playback dashboard containing authentic media control layouts (Play/Pause, Skip, Shuffle, Repeat, Progress Sliders, and Volume controllers).
* **Fluid Micro-Interactions:** Custom transition triggers on content cards that smoothly reveal the signature green Spotify "Play Button" overlay on mouse hover.
* **Refined Typography & Dark Mode Balance:** Precise match of official font hierarchies, text tracking, track labels, and custom translucent hover states against deep application backgrounds.
* **Fully Responsive Flow:** Designed to scale down gracefully, collapsing complex desktop sidebars into an intuitive, mobile-friendly interface.

---

## 🛠️ Tech Stack Used

* **HTML5:** Semantic architecture (`<nav>`, `<aside>`, `<main>`, `<footer>`) to maximize accessibility and SEO readiness.
* **CSS3:** Custom properties (CSS variables), advanced CSS Grid layouts, Flexbox alignments, and backdrop filters.
* **Bootstrap 5 & Icons:** Utilized for structural utility classes and crisp, scalable vector UI components.
* **Modern Typefaces:** Integrated clean sans-serif system type scaling matching native desktop configurations.
