# Personal Portfolio Website
A clean, modern, and fully responsive personal portfolio website developed as part of an internship technical assessment. This project serves as a comprehensive showcase of professional skills, academic projects, technical expertise, and core competencies in front-end web development and software engineering.

## 🚀 Project Overview

The primary objective of this task was to design and implement a structured, performant, and visually engaging single-page portfolio application. The site provides prospective employers, collaborators, and academic evaluators with an interactive overview of my technical profile, project history, and capabilities.

### Key Deliverables
*   **Semantic Structure:** A standard-compliant HTML5 foundational layout.
*   **Modern Styling:** Comprehensive CSS3 design utilizing custom variables, elegant typography, and strict alignment principles.
*   **Interactive Enhancements:** Client-side JavaScript handling dynamic features, seamless scrolling, and robust contact form handling.

---

## ✨ Features & Functionality

### 📱 Responsive & Fluid Layout
*   Built entirely using semantic layout blocks (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`).
*   Implements fluid widths, media queries, and adaptive font sizing to deliver an optimized viewing experience across smartphones, tablets, laptops, and desktop displays.

### 🎨 Design & Theme Consistency
*   Designed around a carefully selected, desaturated color palette (cool slate greys, soft accent highlights, and crisp contrasting backgrounds) that balances professionalism with visual appeal.
*   Strict adherence to the standard CSS Box Model (`box-sizing: border-box`) ensures seamless element padding, spacing, and border layouts without layout breakage.

### 📑 Modular Sections
1.  **Hero / Introduction:** A bold, welcoming top section featuring a concise professional summary and a clear Call-to-Action (CTA) to view recent work or download a resume.
2.  **About Me:** A narrative introduction highlighting academic achievements, career objectives, and specialized technical focuses (e.g., front-end web technologies and enterprise software).
3.  **Skills Inventory:** A cleanly categorized list or grid displaying proficiency across core competencies:
    *   *Front-End:* HTML5, CSS3, JavaScript (DOM Manipulation, Array methods, Event Architecture).
    *   *Core & Enterprise Systems:* Core Java, Object-Oriented Programming (OOP), Multithreading, Database Connectivity (JDBC), and Distributed Architectures (Servlets, RMI, EJB).
4.  **Project Showcase:** An interactive section detailing key academic and personal development milestones. Each entry details the technology stack utilized, core architecture, and measurable outcomes.
5.  **Interactive Contact Section:** A fully functional user contact form with built-in client-side validation to capture messages, queries, or collaboration requests securely.

### ⚙️ Enhanced Interactivity (JavaScript)
*   **Smooth Scroll Navigation:** Dynamic scroll-into-view event handlers linked to navbar links for fluid page transitions.
*   **Active Section Tracking:** Event listeners monitor scrolling behavior to dynamically update the navigation menu highlight based on the user's active view.
*   **Form Validation & Handling:** Pre-submission JavaScript validation checks form data integrity (e.g., verifying email formats and ensuring required fields are non-empty) before simulating or triggering transmission.

---

## 🛠️ Technology Stack & Concepts Applied

This project synthesizes a wide range of core technical competencies:

*   **Structure:** HTML5 (Forms, Structural Elements, Optimized Media Embedding).
*   **Styling & Presentation:** CSS3 (Custom Properties/Variables, Advanced Selectors, Positioning layers, Subtle Hover Transitions, Custom Scrollbars).
*   **Behavioral Layer:** Vanilla JavaScript (ES6+ features, Dynamic DOM manipulation, Event bubbling/capturing, Array transformations).
*   **Architecture & Workflow:** Standard-compliant document tree optimization, cross-browser compatibility handling, and production-ready code formatting.
